# Password Generator 🔐

## Description
A simple Python password generator that creates strong, secure passwords based on user-defined length. It uses Python's built-in `random` and `string` libraries to generate a randomized mix of letters, digits, and special characters.

## Features
- User-defined password length
- Includes uppercase, lowercase, numbers, and symbols
- Uses Python’s `random.choice()` and `string` character sets
- Clean command-line interface

## How to Run
1. Clone or download this repository
2. Open the folder in your IDE or terminal
3. Run the file using:
   ```bash
   python password_generator.py
